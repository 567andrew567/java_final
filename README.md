## 遊玩方式

WASD鍵操作，R重新開始
* 綠色:起點
* 白色:路徑
* 黃色:終點
* 黑色:牆
* 灰色:洞
* 紅色:有bug出事了


## 生地圖說明

讓方塊隨機走生路徑，把不是路徑的區域設成洞或是牆

* 隨機走方式
  * 隨便上下左右走，走過先設成牆
  * 用一個array紀錄路徑
  * 用Block自動判定可不可以走
  * 如果隨機走完還是留在原地就重新隨機這次
  * 避免卡死，如果重複10次都在同一個地方就全部重算
  * 把路徑換成路

* 牆、洞分辨方式，用dfs算相鄰的不是路的格子數
  * <30格設成洞
  * \>=30格設成牆
